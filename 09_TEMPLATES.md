§5. WORKFLOW SCENARIOS
═══════════════════════════════════════════════════════════════════

<workflows>

─── SCENARIO A: "I just need a track" ───
1. Describe your idea in plain words → Quick Start (item 1)
2. Get 2–3 variants
3. Pick the best → copy Style + Lyrics into Suno
4. Generate 3–4 versions in Suno → pick the best
5. (Optional) Extend for a longer track

─── SCENARIO B: "I want it to sound like [artist]" ───
1. Clone Mode (item 3) → name the artist or upload audio
2. System extracts "sonic DNA" (Copyright Safe)
3. Get 2 variants: exact clone + creative interpretation
4. Copy into Suno → generate → iterate

─── SCENARIO C: "Blend two genres" ───
1. Hybrid Lab (item 4) → name two genres
2. System checks Fusion Pairs / Anti-Pairs
3. Suggests bridge genre and Narrative Blend
4. Get prompt with genre distribution across sections

─── SCENARIO D: "I have a photo" ───
1. Gemini Music Lab (item 7) → upload photo + describe genre
2. Get Gemini prompt → generate sketch (Clip: 30 sec / Pro: 3 min)
3. Like it? → Clip: "transfer suno" for full track
              → Pro: already a complete track! Or "transfer suno" for Voices/Studio
4. Don't like it? → iterate ("make it more...")

─── SCENARIO E: "Need a prompt for both platforms" ───
1. Dual Mode (item 8) → describe idea
2. Get Suno format + Gemini Pro format in parallel
3. Both = complete tracks (Suno 5+ min, Gemini Pro 3 min)

─── SCENARIO F: "My prompt isn't working" ───
1. Audit (item 10) → paste your prompt
2. System runs 10-point diagnostic
3. Shows errors + corrected version
4. Copy corrected → regenerate

─── SCENARIO G: "I want a series of tracks in one style" ───
1. Generate the first track (any scenario)
2. Save Style string in Style Library
3. For each new track: same Style + new lyrics
4. Stylistic consistency guaranteed

─── SCENARIO H: "I want a long track in Gemini" (NEW v1.1) ───
1. Gemini Music Lab → Lyria 3 Pro (item 7a, up to 3 min)
2. Need longer? → ProducerAI (item 7g, up to ~5 min)
3. Even longer? → "transfer suno" → Suno + Extend (5+ min)

─── SCENARIO I: "I want to sing with my own voice" (v5.5 · NEW v1.1) ───
1. Suno v5.5 → Voices (item 5e)
2. Upload voice (30s–4min) → complete verification
3. Generate drafts with regular voices (save credits)
4. Final version → with Voices (4 credits/track)

─── SCENARIO J: "I want a signature sound" (v5.5 · NEW v1.1) ───
1. Suno v5.5 → Custom Models (item 5f)
2. Upload 6+ own tracks (with rights!)
3. Model trains in a few minutes
4. Generate with minimal prompt — model "knows" the style
</workflows>
